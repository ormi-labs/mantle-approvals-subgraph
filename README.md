ERC20/ERC721 Token Approvals


## Start graph-node

```
docker-compose up -d
```

ipfs webui: http://127.0.0.1:5001
graph index node playground: http://127.0.0.1:8030/graphql/playground

## Build subgraph

```
yarn build
```

## Create subgraph

```
yarn create-local
```

## Deploy subgraph

```
yarn deploy-local
```

## Get subgraph status

Open graph index node playground and input flowing graphl request

```graphql
{
  indexingStatusForCurrentVersion(subgraphName: "mantle/approvals") {
    synced
    health
    fatalError {
      message
      block {
        number
        hash
      }
      handler
    }
    chains {
      chainHeadBlock {
        number
      }
      latestBlock {
        number
      }
    }
  }
}
```

```json
{
  "data": {
    "indexingStatusForCurrentVersion": {
```
